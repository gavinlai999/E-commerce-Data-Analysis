# **E-Commerce-Sales-Analysis**

## **Project Overview**
- **Project Title:** E-Commerce Sales Analysis  
- **Dataset:** Sample Superstore (Retail Orders Dataset)

![images](https://github.com/user-attachments/assets/bbc0718b-e6e6-4110-a843-bebced1ca3e0)


This project loads and analyzes e-commerce sales data, cleans and transforms it using pandas, and performs exploratory analysis to uncover key business insights. It serves as a practical walkthrough of data wrangling, EDA, feature engineering, and visualization inside a single notebook.

---

## **Project Steps**

### **1. Set Up the Environment**
- **Tools Used:** Jupyter Notebook, Python  
- **Goal:** Create a clean analysis environment with organized folders to support smooth data exploration and reproducible work

---

### **2. Load the Dataset**
- **Source:** Sample Superstore dataset (commonly used for Tableau and retail analytics)  
- **Storage:** Place the CSV inside a `data/` folder or load it directly into the notebook
